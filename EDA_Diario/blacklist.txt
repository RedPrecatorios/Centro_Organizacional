# ─────────────────────────────────────────────────────────────────────────────
# BLACKLIST — Casos que nao devem receber contato
# ─────────────────────────────────────────────────────────────────────────────
# Como preencher:
#   TIPO | VALOR | MOTIVO (motivo e opcional)
#
# Tipos aceitos:
#   CPF      -> bloqueia a pessoa inteira (todos os contatos/emails)
#   NOME     -> bloqueia a pessoa inteira pelo nome
#   TELEFONE -> bloqueia apenas aquele numero
#   EMAIL    -> bloqueia apenas aquele email
#
# Linhas que comecam com # sao ignoradas (comentarios)
# ─────────────────────────────────────────────────────────────────────────────

# Exemplos (remova o # para ativar):
# CPF      | 12345678900          | Solicitou nao receber contato
# NOME     | JOAO DA SILVA        | Pessoa falecida
# TELEFONE | 11999990000          | Numero invalido
# EMAIL    | exemplo@email.com    | Email com bounce permanente
