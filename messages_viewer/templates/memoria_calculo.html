Memória de Cálculo

Pesquise por requerente (nome) ou por processo e incidente — um modo de cada vez. Até carregar, tudo fica em zero.

{% if not memoria_mysql_configured %}

MySQL não configurado. Defina MEMORIA_MYSQL_DATABASE (e, se necessário, MEMORIA_MYSQL_HOST, MEMORIA_MYSQL_USER, MEMORIA_MYSQL_PASSWORD) no .env para ativar a pesquisa. A memória fica em zero até carregar um registo.

{% else %}

Como funciona. Use ou o nome do requerente, ou o nº do processo com o nº do incidente (não os dois). Os totais e as sugestões de proposta só surgem após pesquisar e carregar (ou escolher entre vários, se existirem). Antes disso, tudo fica em zero.

{% if not calculo_atualizacao_configured %}

API de cálculo não configurada. Defina CALCULO_ATUALIZACAO_API_URL no .env da plataforma (ex.: http://127.0.0.1:5099, igual ao CALCULO_ATUALIZACAO_API_HOST/PORT do serviço interno) e arranque a unidade systemd; o botão Atualizar Cálculo fica inactivo até isso.

{% endif %} {% if not analise_processual_configured %}

Análise processual indisponível. Confirme REFACTOR_TJSP no servidor (REFACTOR_TJSP_PATH e venv com dependências).

{% endif %} {% endif %}
